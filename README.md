## Notebook Descriptions

### Part 1 — Heart Disease Prediction

**`Part1_Q1.ipynb`** — Exploratory Data Analysis & Preprocessing

Loads the Cleveland Heart Failure dataset and performs full EDA: feature distributions, class balance, cross-tabulations (Sex × HeartDisease, ChestPainType × HeartDisease), and missing value analysis. Imputes zero-cholesterol values with KNN imputation (fit on training data only), applies StandardScaler to numerical features and PowerTransformer to the skewed `Oldpeak` column, and one-hot encodes categorical variables. Saves the preprocessed splits to parquet for downstream notebooks.

**`Part1_Q2.ipynb`** — Logistic Lasso Regression

Fits an L1-regularized Logistic Regression on the preprocessed features. Reports performance metrics (F1, balanced accuracy, AUC-ROC) and visualizes coefficient magnitudes as a bar chart to interpret which features the model relies on. Also explores pairwise interaction features and grid-search hyperparameter tuning.

**`Part1_Q3.ipynb`** — MLP + SHAP Explanations

Trains a two-hidden-layer MLP (64→32, ReLU, Adam, early stopping) and evaluates it on the test set. Uses SHAP KernelExplainer to compute Shapley values for all test samples, then visualizes: individual waterfall plots for two positive and two negative predictions, and a global beeswarm summary plot showing overall feature importance.

**`Part1_Q4.ipynb`** — Neural Additive Models (NAM)

Implements a Neural Additive Model in PyTorch: each feature is passed through its own small neural network and contributions are summed. Trains with hyperparameter search over hidden sizes and dropout rates (validated by AUROC). Plots per-feature contribution curves to provide inherently interpretable feature importance.

---

### Part 2 — Chest X-Ray Classification

**`Part2_Q1_Q2.ipynb`** — EDA + CNN Classifier

Explores the Kaggle Chest X-Ray dataset (label distribution, qualitative examples, class imbalance analysis). Defines preprocessing: resize to 224×224, grayscale-to-RGB conversion, ImageNet normalization, and data augmentation (random flips, rotations, color jitter) for training. Fine-tunes a pretrained **ResNet-18** with a replaced classification head (binary: NORMAL vs PNEUMONIA), using weighted cross-entropy loss to handle class imbalance. Saves the best checkpoint as `best_model.pth` and preprocesses all splits to `.npz` files for faster loading.

**`Part2_Q3.ipynb`** — Integrated Gradients

Implements Integrated Gradients from scratch using a black-pixel baseline (zero raw pixel values, mapped to normalized space). Visualizes attribution maps for 5 NORMAL and 5 PNEUMONIA test samples. Investigates baseline sensitivity by comparing attributions across four different baselines (black image, zero-in-normalized-space, white image, Gaussian noise).

**`Part2_Q4.ipynb`** — Grad-CAM

Implements Gradient-weighted Class Activation Mapping (Grad-CAM) by hooking into the final convolutional layer of ResNet-18. Visualizes heatmaps overlaid on 5 NORMAL and 5 PNEUMONIA test samples, and compares findings with the Integrated Gradients results from Q3.

**`Part2_Q5.ipynb`** — Data Randomization Test

Implements the sanity check from *Adebayo et al., "Sanity Checks for Saliency Maps"*. Retrains ResNet-18 on the training set with randomly permuted labels (fixed seed for reproducibility) and saves the result as `best_model_shuffled.pth`. Runs both Grad-CAM and Integrated Gradients on the randomized model and compares attribution maps against the correctly-trained model to verify that the saliency methods are sensitive to learned parameters rather than acting as input-space edge detectors.

## Hardware Notes

- Part 1 notebooks run on CPU in under a few minutes.
- Part 2 notebooks automatically detect and use **MPS** (Apple Silicon), **CUDA** (NVIDIA GPU), or fall back to CPU. Training `Part2_Q1_Q2.ipynb` and `Part2_Q5.ipynb` (10 epochs each) takes ~5–15 min on GPU and significantly longer on CPU.
- Preprocessed `.npz` files (`data/preprocessed/`) are generated once by `Part2_Q1_Q2.ipynb` and reused by Q4 and Q5 to avoid repeated image loading.
